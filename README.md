# 🌍 Language Translation Tool

## 📌 Project Overview

The Language Translation Tool is an AI-powered web application developed using React.js. It allows users to translate text from one language to another using an online Translation API. The application provides a simple and user-friendly interface where users can enter text, select source and target languages, and instantly view the translated result.

This project was developed as part of the **CodeAlpha Artificial Intelligence Internship**.

---

## 🚀 Features

- Translate text between multiple languages
- User-friendly interface
- Source and target language selection
- AI-powered translation using Translation API
- Real-time translation results
- Responsive React-based frontend

---

## 🛠️ Technologies Used

- React.js
- JavaScript (ES6)
- HTML5
- CSS3
- Fetch API
- MyMemory Translation API

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/YourUsername/CodeAlpha_LanguageTranslationTool.git
```

2. Open the project folder

```bash
cd CodeAlpha_LanguageTranslationTool
```

3. Install dependencies

```bash
npm install
```

4. Start the project

```bash
npm start
```

The application will run at:

```
http://localhost:3000
```

---

## 📖 How to Use

1. Enter text in the input box.
2. Select the source language.
3. Select the target language.
4. Click the **Translate** button.
5. The translated text will appear below.
